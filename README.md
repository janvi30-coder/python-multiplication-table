# Multiplication Table Generator

## Project Description

This Python program generates the multiplication table of any number entered by the user.

## Features

* Takes user input
* Uses a for loop
* Displays multiplication table from 1 to 10

## Concepts Used

* Variables
* User Input
* for Loop
* range()
* print()

## Code

python
num = int(input("Enter the number:"))

for number in range(1,11):
    print(num, "*", number, "=", num * number)
